# Support

If you need help with Event Monitor, use the support ticket format below. Structured tickets let us triage faster and deliver accurate fixes or workarounds.

## Preferred support channel

- Open a GitHub Issue: [Create Support Ticket](https://github.com/pawelca/EventMonitor2.0/issues/new?template=support_request.md)
- Fallback email: [pawel.ciesla89@gmail.com](mailto:pawel.ciesla89@gmail.com)

## Before you submit

- Reproduce the issue at least once.
- Check basic triage in [Troubleshooting & FAQ](./faq).
- Collect Console logs and Event Monitor export if possible.
- Prepare minimal reproduction steps.

## What makes a high-quality ticket

- Clear reproduction steps.
- Expected vs actual behavior.
- Complete environment metadata.
- Artifacts attached (logs/export/screenshot).
- Minimal repro whenever possible.
